# PersonalPortfolio_website
This is a personal portfolio website of Gowtham V R, a BCA student passionate about front-end development. The website showcases personal information, skills, and projects, and provides contact links including LinkedIn and GitHub.

Technologies Used:

- HTML5:- structure and content

- CSS3:- styling, layout, and animations

JavaScript:- scroll animations and interactivity

Features:

- Responsive navigation bar

- Hero section with introduction

- About section highlighting personal background

- Skills section including HTML, CSS, JavaScript, React, C, C++, Python, and MySQL

- Projects section showcasing personal web projects

- Contact section with email, phone, LinkedIn, and GitHub links

- Smooth scroll animations for skills and projects
